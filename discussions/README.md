## üéØ How to Participate

### Starting a Discussion
1. Choose the appropriate category for your topic
2. Search existing discussions to avoid duplicates
3. Use clear, descriptive titles
4. Provide context and details in your post
5. Tag relevant topics using labels

### Discussion Etiquette
- **Be respectful**: Treat all community members with respect
- **Stay on topic**: Keep discussions relevant to the category
- **Search first**: Check if your question has been answered
- **Share knowledge**: Help others when you can
- **Provide context**: Include relevant details about your environment
- **Follow up**: Update discussions with solutions you find

### Best Practices
- **Use markdown formatting** for code snippets and structured content
- **Include examples** when discussing data patterns or configurations
- **Sanitize sensitive data** before sharing logs or configurations
- **Reference documentation** when applicable
- **Mark solutions** that resolve your questions
